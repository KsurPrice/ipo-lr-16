# 🚲 BikeShop - Магазин велоаксессуаров

## 📌 О проекте

**BikeShop** - это веб-приложение, разработанное на фреймворке Django в рамках лабораторной работы №16. 
Приложение представляет собой интернет-магазин аксессуаров для велосипедов с красивым современным дизайном.

---

## 🎯 Цель лабораторной работы

- Освоить процесс создания веб-приложения с использованием фреймворка Django
- Научиться базовой настройке проекта и приложений
- Создать функциональный прототип интернет-магазина

## 🗺️ Структура страниц

### 1. Главная страница (`/`)
- Приветственное сообщение
- Анимированный градиентный фон
- Кнопки навигации к другим страницам
- Адаптивный дизайн

### 2. Страница об авторе (`/about/`)
- **ФИО:** Карась Илья
- **Группа:** 88ТП
- **Курс:** 2
- **Специальность:** Техник-программист
- Контактная информация
- Информация об увлечениях и хобби

### 3. Страница о магазине (`/store-info/`)
- **Тема:** Магазин аксессуаров для велосипедов
- **Ассортимент:**
  - 🔦 Велосипедные фары и фонари
  - 🔒 Велосипедные замки и тросы
  - 🎒 Велосипедные сумки и рюкзаки
  - 🪖 Защитные шлемы
  - 🔧 Инструменты и насосы
  - 💡 Дополнительные аксессуары
- Ссылка на сайт-источник: [citybikes.by](https://citybikes.by/)
- Информация о преимуществах магазина
